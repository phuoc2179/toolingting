Contact → Portal Documents Report

Fetches all contacts for a firm, then maps each contact to the portal documents they can access.

Runs the investment communication-matrix export to resolve documents shared by Investments. Only available on firms that have the investment matrix enabled. Leave off if unsure — any investment-shared documents will simply be flagged in the report.

Special case for firms without an investment matrix: each investment is mapped to its investment entity and fund legal entity (via the investments export), then contacts are found through the IE × FLE × communication-type matrix instead. When on, this takes priority over the investment matrix for Investments-shared documents.

Credentials stay in this page — only sent to the server above.
CORS / local server note
If opening this file directly from disk (file://), browsers block cross-origin requests.
Fix: run python3 -m http.server 8000 in the same folder and open http://localhost:8000/anduin-portal-documents.html
